# Capital Advocates, LLC — Website Redesign

**OIM 3690 Web Technologies · Mini Project #1**

## Client

Bill & Kim Teator — co-founders of Capital Advocates, LLC, a strategic communication, change management, and government relations consulting firm based in Saratoga Springs, NY. Founded in 2003.

## About the Site

A 6-page redesign of capitaladvocates.com, replacing an outdated GoDaddy template with a modern, professional, and conversion-oriented website. The site positions Capital Advocates for high-stakes clients — corporate executives, non-profit leaders, and government officials — who need to quickly verify the firm's credibility and initiate a consultation.

**Pages:**
- `index.html` — Home
- `about.html` — About Us (principal bios)
- `services.html` — Services (4 core areas + social media & video)
- `case-studies.html` — Our Impact (4 case studies)
- `insights.html` — Insights & Resources
- `contact.html` — Contact Us

## Tech Stack

- Semantic HTML5
- External CSS (`styles.css`) with CSS custom properties, Flexbox, and Grid
- Vanilla JavaScript (`main.js`) — scroll reveal, mobile nav, form handling
- Google Fonts (Playfair Display + Lato)
- No frameworks or libraries

## What I Learned

- How to translate a real client brief into a structured, multi-page site architecture
- Using CSS custom properties (variables) to maintain a consistent brand palette across 6 pages
- Implementing `IntersectionObserver` for scroll-triggered reveal animations without JavaScript libraries
- Designing responsive layouts with CSS Grid that gracefully collapse on mobile
- Writing semantic HTML with proper ARIA labels and `alt` text for accessibility
